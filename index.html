Python Basics Tutorial

1. Data Types

Python has various data types including:

Integer: 5, -3
Float: 3.14, -0.5
String: "hello"
Boolean: True, False
List: [1, 2, 3]
Tuple: (4, 5, 6)
Dictionary: {"name": "Alice"}

Tasks

colors = ["blue", "green", "red"]
print(colors)

2. If-Else Statements

age = 18
if age >= 18:
    print("You can vote!")
else:
    print("Can't vote yet.")

Tasks

num = -5
if num > 0:
    print("Positive")
elif num < 0:
    print("Negative")
else:
    print("Zero")

3. For Loop

for fruit in ["apple", "banana"]:
    print(fruit)

Tasks

total = 0
for i in range(1, 11):
    total += i
print(total)
